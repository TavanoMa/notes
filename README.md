# DevNotes

Aplicação de notas desenvolvida para criação, edição e organização de anotações diretamente no navegador, com foco em simplicidade, produtividade e responsividade.

## Funcionalidades
- Criar novas notas  
- Editar notas existentes  
- Excluir notas  
- Fixar (pin) notas importantes  
- Pesquisar notas por texto  
- Interface totalmente responsiva  
- Persistência automática dos dados  

## Responsividade
A aplicação é totalmente **responsiva**, adaptando-se corretamente a desktops, tablets e dispositivos móveis, garantindo boa usabilidade em qualquer tamanho de tela.

## Armazenamento de Dados
Todas as notas são armazenadas localmente no navegador por meio do **LocalStorage**, mantendo os dados salvos mesmo após recarregar ou fechar a página.

## Tecnologias Utilizadas
- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- LocalStorage
